# Colores

Catálogo de colores disponibles para los productos.

**Operaciones disponibles:**
- Buscar colores
- Sincronizar con SAP
- Establecer valor por defecto

| Campo | Descripción |
|-------|-------------|
| Código | Código único del color |
| Descripción | Nombre del color |
| Valor por defecto | Indica si es el color predeterminado |

{% hint style="info" %}
La sincronización descarga los datos actualizados desde SAP al sistema.
{% endhint %}

{% hint style="success" %}
El valor por defecto se seleccionará automáticamente en los formularios de creación.
{% endhint %}
