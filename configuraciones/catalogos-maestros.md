# Catálogos Maestros

Los catálogos maestros contienen los datos base que alimentan los formularios del sistema.

{% hint style="info" %}
**Ruta:** Menú → Configuraciones → [Seleccionar opción]
{% endhint %}

## Operaciones Comunes

Todos los catálogos comparten estas operaciones básicas:

**Buscar Registros**

1. Use el campo "Ingrese código, descripción"
2. La tabla se filtrará automáticamente
3. Los resultados se mostrarán en tiempo real

**Sincronizar con SAP**

1. Haga clic en el botón **"Sincronizar con SAP"**
2. Confirme la acción en el mensaje emergente
3. Espere a que se complete la sincronización

{% hint style="info" %}
La sincronización descarga los datos actualizados desde SAP al sistema.
{% endhint %}

**Establecer Valor por Defecto**

1. Localice el registro deseado en la tabla
2. Haga clic en **"Aplicar por defecto"**
3. Confirme la acción

{% hint style="success" %}
El valor por defecto se seleccionará automáticamente en los formularios de creación.
{% endhint %}
