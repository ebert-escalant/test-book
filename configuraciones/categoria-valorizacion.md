# Categoría de Valorización

Catálogo de categorías de valorización para los productos.

**Operaciones disponibles:**
- Buscar categorías
- Sincronizar con SAP
- Establecer valor por defecto

| Campo | Descripción |
|-------|-------------|
| Código | Código de la categoría |
| Descripción | Nombre de la categoría |
| Valor por defecto | Indica si es la categoría predeterminada |

{% hint style="info" %}
La sincronización descarga los datos actualizados desde SAP al sistema.
{% endhint %}

{% hint style="success" %}
El valor por defecto se seleccionará automáticamente en los formularios de creación.
{% endhint %}
