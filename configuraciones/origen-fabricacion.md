# Origen Fabricación

Catálogo de orígenes de fabricación para los productos.

**Operaciones disponibles:**
- Buscar orígenes
- Sincronizar con SAP
- Establecer valor por defecto

| Campo | Descripción |
|-------|-------------|
| Código | Código del origen |
| Descripción | Nombre del origen |
| Valor por defecto | Indica si es el origen predeterminado |

{% hint style="info" %}
La sincronización descarga los datos actualizados desde SAP al sistema.
{% endhint %}

{% hint style="success" %}
El valor por defecto se seleccionará automáticamente en los formularios de creación.
{% endhint %}
