# Proveedores y Paquetes

## Proveedores

Catálogo de proveedores del sistema.

**Ruta:** Configuraciones → Proveedor

| Campo | Descripción |
|-------|-------------|
| Código | Identificador |
| Descripción | Nombre del proveedor |
| Abreviatura | Código corto |
| Estado | Activo / Inactivo |
| Fecha de registro | Fecha de creación |

![Proveedores](../.gitbook/assets/proveedores.png)

### Crear Proveedor

1. Haga clic en **"Nuevo"**
2. Complete los campos requeridos
3. Haga clic en **"Guardar"**

### Editar Proveedor

1. Haga clic en el ícono **Editar** (✏️)
2. Modifique los datos
3. Haga clic en **"Actualizar"**

## Paquetes

Catálogo de tipos de paquetes.

**Ruta:** Configuraciones → Paquete

| Campo | Descripción |
|-------|-------------|
| Descripción | Nombre del paquete |
| Abreviación | Código corto |
| Estado | Activo / Inactivo |
| Fecha de registro | Fecha de creación |

![Paquetes](../.gitbook/assets/paquetes.png)

{% hint style="info" %}
Los paquetes se asocian a las órdenes de compra para identificar el tipo de empaque.
{% endhint %}
