# Unidad de Medida

Catálogo de unidades de medida para los productos.

**Operaciones disponibles:**
- Buscar unidades
- Sincronizar con SAP
- Establecer valor por defecto

| Campo | Descripción |
|-------|-------------|
| Código | Código de la unidad |
| Descripción | Nombre de la unidad |
| Valor por defecto | Indica si es la unidad predeterminada |

{% hint style="info" %}
La sincronización descarga los datos actualizados desde SAP al sistema.
{% endhint %}

{% hint style="success" %}
El valor por defecto se seleccionará automáticamente en los formularios de creación.
{% endhint %}
